SeanR

SeanR is where I put (mostly) all of the free packages I install on my phone. No ads, no paid packages, no monetization, no BS.

Some tweaks work only on older versions of apps, which you may downgrade to by installing AppStore++ (by CokePokes) from here.

The packages here are all being simultaneously run on my iPhoneXR with iOS 14.4.1 jailbroken with Unc0ver+Fugu14, refer to the author of any given tweak you have questions about the compatibility of, as some of this information is not present in its entirety on this repo.
